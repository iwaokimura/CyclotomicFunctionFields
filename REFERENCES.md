# References

## Primary References

### Hayes' Original Papers

1. **David R. Hayes** (1974). "Explicit Class Field Theory for Rational Function Fields."
   *Transactions of the American Mathematical Society*, 189: 77-91.
   - DOI: 10.2307/1996857
   - The foundational paper for this formalization

2. **David R. Hayes** (1979). "Explicit Class Field Theory in Global Function Fields."
   *Studies in Algebra and Number Theory*, pp. 173-217.

### Carlitz Module Theory

3. **Leonard Carlitz** (1935). "On Certain Functions Connected with Polynomials in a Galois Field."
   *Duke Mathematical Journal*, 1(2): 137-168.
   - DOI: 10.1215/S0012-7094-35-00114-4
   - Original development of the Carlitz module

4. **David Goss** (1996). *Basic Structures of Function Field Arithmetic*.
   Springer-Verlag, Berlin.
   - ISBN: 978-3-540-61087-8
   - Comprehensive modern treatment

## Function Field Arithmetic

5. **Michael Rosen** (2002). *Number Theory in Function Fields*.
   Graduate Texts in Mathematics, Vol. 210. Springer.
   - ISBN: 978-0-387-95335-8
   - Excellent introduction to the subject

6. **Dino Lorenzini** (1996). *An Invitation to Arithmetic Geometry*.
   Graduate Studies in Mathematics, Vol. 9. AMS.
   - ISBN: 978-0-8218-0267-9

8. **Serge Lang** (1994). *Algebraic Number Theory* (2nd ed.).

## Additive Polynomials and Drinfeld Modules
